# Advent of Code
In Rust.

Setup for day `1` of year `2020`:

```bash
./setup 2020 1
source .env
```

## Commands
To set up a new day, run:

```bash
make today
```

To run today's solver:

```bash
make run
```

To save today once the solution is complete (to commit to git):

```bash
make save
```
